# SEN Group Landing Page

Современный лендинг для SEN Group — экосистемы доверия и управления будущего.

## О проекте

SEN Group — это технологический и финтех-холдинг, объединяющий несколько компаний:
- **SenDigital** — цифровая трансформация и разработка
- **SenTechAudit** — технологический аудит
- **SenFinance** — финтех решения
- **SenConsulting** — стратегический консалтинг
- **SenMedia** — медиа и коммуникации
- **FoundationHub** — социальный эффект и прозрачность

## Технологии

- **Next.js 14** — React фреймворк
- **TypeScript** — типизация
- **Tailwind CSS** — стилизация
- **Framer Motion** — анимации
- **Inter Font** — типографика

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен-версии
npm start
```
