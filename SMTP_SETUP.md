# Настройка SMTP для отправки писем

## Переменные окружения в Netlify

Для работы отправки писем через форму необходимо настроить следующие переменные окружения в Netlify:

1. Перейдите в настройки вашего сайта на Netlify: **Site settings → Environment variables**
2. Добавьте следующие переменные:

```
SMTP_USER=info@sengroup.one
SMTP_PASSWORD=ваш_пароль_от_почтового_ящика
```

## Настройки SMTP

- **SMTP сервер:** pkz32.hoster.kz
- **SMTP порт:** 465 (SSL)
- **Отправитель:** info@sengroup.one
- **Получатель:** info@sengroup.one

## Проверка работы

После настройки переменных окружения форма будет отправлять письма на указанный адрес.
